Learning Tracker React App

The React application should be running at:

  • http://localhost:5173
  • http://localhost:5174

Features Included:

  • ✅ Knowledge Tree Visualization
  • ✅ Spaced Repetition Flashcards (SM-2 Algorithm)
  • ✅ Study Sessions (Traditional & Swipeable Deck)
  • ✅ Bidirectional Tree ↔ Cards Synchronization
  • ✅ Progress Tracking
  • ✅ Multi-select Operations
  • ✅ Undo/Redo Functionality
  • ✅ Search & Filtering
  • ✅ Local Storage Persistence
  • ✅ Modern React Architecture with TypeScript
  • ✅ Framer Motion Animations
  • ✅ Responsive Design

How to Use:

  1. Click on the Knowledge Tree tab to organize your learning topics
  2. Add nodes by clicking the + button
  3. Switch to Cards tab to study with spaced repetition
  4. Use the traditional study session or swipeable card deck
  5. Rate your performance to optimize review intervals
